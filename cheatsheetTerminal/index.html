Les commandes console linux

Fichiers

  • mkdir mondossierCrée un dossier
  • rm -r mondossierSupprime un dossier
  • rm fichierSupprime un fichier
  • mv monfichier mondossierDéplace un fichier dans un dossier
  • cp monfichier mondossierCopie un fichier dans un dossier

Navigation et visualisation

  • pwdAffiche le chemin du dossier
  • lsListe le contenu du dossier
  • cd mondossierSe déplace vers un dossier
  • cd ..Retourne au dossier parent
  • cdRetourne à la racine
  • clearNettoie la fenêtre
  • cd ~Le caractère "~" représente le répertoire HOME de l'utilisateur

Divers

  • sudo apt-get installInstalle un programme depuis internet
  • dpkg -iInstalle un programme depuis un fichier deb
  • sudo apt-get updateEffectue la mise à jour des dépôts
  • sudo apt-get upgradeEffectue la mise à jour des paquets
  • find mondossier -name "monfichier"Recherche un fichier dans un dossier
  • wget URLtéléchargement de fichiers depuis le web
